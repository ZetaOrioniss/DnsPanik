
Description de l'outil:

Le programme est un outil codé en Python-3.11.2 permettant de lister les répertoires et/ou les sous-domaines existants d'un site web saisis par l'utilisateur. Ce dernier doit préciser la wordlist à utiliser pour effectuer le scan. Chaque site web scanné doit être enregistré dans une base de donnée ainsi que le statut et le type de scan effectué (répertoire/sous-domaine).
La base pourra servir de sauvegarde également, afin de reprendre un scan là où il s'est arrété.
Enfin, on pourra rechercher si un élement précis (répertoire/sous-domaine) existe dans la base afin d'en ressortir les résultats de scan. S'il n'y pas de résultats, et sur volonté de l'utilisateur, un scan est effectué avec l'élément désiré et les résultats sont ressortis et enregistré dans la base.


Entitées:

    -Les domaines
    -Les sous-domaines
    -Les répertoires
    -Les statuts des scans
    -L'état des domaines


1/ Listes des Tâches nécessaires

    - Gestion des domaines analysés / en cours de scan:

            -Il faut enregistrer/supprimer un domaine
            -Il faut enregistrer/modifier le statut du scan (finis / en cours) pour un domaine

    - Enregistrer les répertoires d'un domaine:

            -IL faut enregistrer le répertoires testé
            -Il faut préciser si le répertoires testé est existants ou non (état)
   
    - Enregistrer les sous-domaines d'un domaine:

            -Il faut enregistrer un domaine
            -Il faut enregistrer le sous-domaine testé
            -Il faut préciser si le sous-domaine testé est existants ou non (état)

    -Supprimer un domaine:

            -Il faut supprimer toutes les données dépendantes de ce domaine.

2/ Description des entitées:

    Domaine: c'est l'url du site web ciblé
    Répertoire: c'est un dossier constituant l'architecture du site web
    Sous-domaine: c'est le sous-domaine du site web
    État du domaine: c'est l'existance du Domaine
