

Implémentation d'une BDD:

    Créer une fonction qui vérifie que la base de donnée existe
            - Si elle n'existe pas, on la crée
    
    Créer une fonction qui crée les tables (Domains, Subdomains, Directories) avec le nom des colonnes adéquates
    
    Créer une fonction qui ajoute les données sur la table Domains (sera appelée par req_subdomain)
    
    Créer une fonction qui ajoute les données sur la table Directories (sera appelée par req_directories)
